Ecommerce Analysis Project

Overview
This project analyzes ecommerce transactional data to generate actionable business insights on sales, customers, and products.

Database Structure
- customers
- products
- orders
- order_items

Key Business Questions Answered
1. Top products by revenue and quantity sold
2. Monthly revenue trends and growth
3. High-value customers analysis
4. Daily sales performance with running totals
5. Customers at risk of churn

SQL Concepts Used
- Common Table Expressions (CTEs)
- Window functions (RANK, LAG, PERCENT_RANK)
- Aggregations and joins
- Time-based analysis

How to Run
1. Execute `schema.sql`
2. Run `sample_data.sql`
3. Execute queries from `analysis_queries.sql`
