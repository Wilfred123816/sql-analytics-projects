Employee Analysis Project

Overview
This project focuses on workforce and compensation analytics using structured employee and department data.

Database Structure
- departments
- employees

Key Business Questions Answered
1. Average salary and budget utilization by department
2. Top earners in each department
3. Salary comparison against department averages

SQL Concepts Used
- Advanced CTE chains
- Window functions
- Ranking and percentiles
- Statistical analysis (STDDEV, Z-scores)

How to Run
1. Execute `schema.sql`
2. Run `sample_data.sql`
3. Execute queries from `analysis_queries.sql`
